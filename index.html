Insight Utama
Sebanyak 376 baris data memiliki nilai 0 tidak logis pada fitur vital, memerlukan class-conditional imputation agar model tidak bias.
Eksperimen 2 membuktikan target leakage mendongkrak performa semu hingga 96,30% ROC-AUC, yang berbahaya untuk diagnosis klinis.
Optuna tuning pada nested CV bebas leakage berhasil menaikkan ROC-AUC menjadi 77,33% secara valid untuk deployment nyata.
