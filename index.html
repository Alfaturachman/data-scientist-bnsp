Temuan Utama & Analisis Eksperimen

Sebanyak 376 baris data memiliki nilai 0 tidak logis pada fitur vital, memerlukan class-conditional imputation agar model tidak bias.

Eksperimen 2 membuktikan target leakage mendongkrak performa semu hingga 96,86% ROC-AUC, yang berbahaya untuk diagnosis klinis.

Tuning Optuna pada LightGBM bebas leakage berhasil mencapai 75,32% Accuracy secara valid untuk deployment nyata.
